1. Async không phải cứ thêm async là nhanh

I/O-bound

Dùng async def cho database driver async, HTTP client async và hàng đợi.

CPU-bound

Đưa việc nặng sang worker, queue hoặc process pool; đừng khóa event loop.

from httpx import AsyncClient

async def fetch_profile(user_id: str):
    async with AsyncClient(timeout=5) as client:
        r = await client.get(f"https://profile.internal/{user_id}")
        r.raise_for_status()
        return r.json()

2. Database: session ngắn, query rõ, pool biết giới hạn

Một lỗi production rất phổ biến là để request giữ connection quá lâu. Hãy dùng dependency tạo session theo request, đóng đúng lúc, và quan sát pool saturation.

Quy tắc gọn: API layer không biết transaction phức tạp; service layer gom nghiệp vụ; repository layer chịu trách nhiệm query.

3. Bảo mật theo lớp

  • Validate input bằng Pydantic model chặt chẽ.
  • Rate limit theo IP, user và API key.
  • Log request id, không log secret hoặc token.
  • Áp dụng CORS tối thiểu thay vì wildcard.

4. RAG endpoint cần hàng rào riêng

Với endpoint phục vụ LLM/RAG, hãy tách quota, giới hạn kích thước prompt, sanitize document chunk, và ghi lại trace để debug hallucination.
